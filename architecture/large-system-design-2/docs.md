# 1장. 근접성 서비스

- 근접성 서비스(proximity service) 설계

## 1단계: 문제 이해 및 설계 범위 확정

- 기능 요구사항
    - 검색 반경
    - 최대 허용 반경
    - UI 검색 반경 변경 기능
    - 정보의 추가, 삭제, 갱신 방안
    - 실시간 업데이트 여부
    - 화면 자동 갱신 여부
- 비기능 요구사항(non-functional requirements):
    - 낮은 응답 지연
    - 데이터 보호(아래는 사용자 위치 정보 관련 보호 법안)
        - GDPR
        - CCPA
    - 고가용성 및 규모 확장성
- 개략적 규모 추정:
    - QPS 계산
